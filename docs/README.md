# Showcase y Pages

Esta carpeta esta preparada para publicarse con GitHub Pages usando `/docs` como fuente.

## Contenido

- `index.html`: galeria principal para escuchar todos los ejemplos
- `report/`: memoria, propuesta y figuras
- `showcase/assets/`: todos los archivos `MIDI` y sus previews `MP3`

## Activacion

1. Ve a `Settings > Pages` en el repositorio.
2. Selecciona `Deploy from a branch`.
3. Elige `main` y la carpeta `/docs`.
4. Guarda.

Una vez activado, la galeria quedara servida por GitHub Pages.
